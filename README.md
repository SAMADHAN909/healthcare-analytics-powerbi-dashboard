# Healthcare Data Analytics Project (SQL | Excel | Power BI)

This is an end-to-end healthcare data analytics project built using SQL, Excel, and Power BI.
The project focuses on patient analysis, hospital performance, and revenue insights using
industry-level KPIs and dashboard design.

## Dataset Overview
Healthcare dataset contains the following columns:
- Name
- Age
- Gender
- Blood Type
- Medical Condition
- Date of Admission
- Doctor
- Hospital
- Insurance Provider
- Billing Amount
- Room Number
- Admission Type
- Discharge Date
- Medication
- Test Results

## Tools & Technologies
- SQL (Data Cleaning, KPI Queries, Analysis)
- Excel (Data Cleaning, Pivot Tables, Basic Analysis)
- Power BI (Dashboard, DAX, Visualizations)

## SQL Analysis
- Total Patients & Emergency Patients
- Average Age Calculation
- Admission Type Analysis
- Gender & Blood Group Analysis
- Medical Condition-wise Patient Count
- Test Result Classification
- Top Revenue Hospitals & Doctors
- Insurance-wise Billing Analysis

## Excel Analysis
- Data Cleaning & Formatting
- Pivot Tables for KPI Analysis
- Charts for Trend Analysis
- Summary Reports

## Power BI Dashboard
- KPI Cards (Total Patients, Avg Age, Emergency Patients)
- Admission Type Distribution
- Gender & Blood Group Charts
- Medical Condition Analysis
- Test Results Overview
- Revenue Analysis (Hospital, Doctor, Insurance)

## Project Objective
To demonstrate practical data analytics skills using SQL, Excel, and Power BI
with real-world healthcare data and industry-level KPIs.
